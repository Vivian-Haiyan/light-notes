# 快速切页与默认范例数据设计

## 目标

1. 保留当前所有页面布局不变。
2. 提升左侧栏页面切换的响应速度。
3. 为首次进入且暂无内容的用户恢复默认范例数据。

## 性能方案

- 书架页不再对每本书逐一读取笔记，改为一次性读取当前用户的全部笔记后在前端分组。
- 灵感页不再对每条灵感逐一读取来源书籍，改为一次性读取书籍后通过 `book_id` 本地映射标题。
- 背景与动效样式保持不变，优化只发生在数据读取方式。

## 默认范例数据

- 仅当用户云端书籍为空时，初始化一次默认数据。
- 默认书籍沿用最初界面中的示例：
  - 百年孤独
  - 月亮与六便士
  - 小王子
  - 瓦尔登湖
  - 悉达多
- 默认笔记沿用最初界面风格，覆盖 `感悟`、`金句`、`书摘`、`灵感` 等标签，使书架、灵感集锦、标签管理、数据统计都能立即展示完整示例。
- 初始化完成后写入本地标记，之后不再重复灌入。

## 边界

- 不改变现有视觉布局。
- 不覆盖用户已有数据。
- 用户删除默认内容后，不会再次自动出现。
