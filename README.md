# 拾光札记 · LightNotes

> 在光影与文字之间，收藏每一次阅读心动。

**拾光札记** 是一款风格清新治愈的读书笔记与知识管理 Web 应用。  
它帮助你管理书架、摘录金句、整理课堂笔记、收集灵感，并提供阅读计划和数据统计，让阅读与学习变得更有温度。

---

## ✨ 主要功能

- 📚 **书架管理**：添加书籍、编辑信息、导入本地书单（JSON/CSV），支持多维度筛选与批量导出。
- 📝 **笔记与书摘**：为每本书记录笔记，支持 Markdown 编辑、标签分类、原文引用，快速捕捉金句与感悟。
- 💡 **灵感集锦**：自动聚合“灵感”“想法”“思考”类笔记，以便签墙形式展示，支持分类筛选与排序。
- 📅 **阅读计划**：制定阅读计划，跟踪进度，养成阅读习惯。
- 🏷️ **标签系统**：自由创建标签，按标签浏览笔记，支持重命名与合并。
- 📊 **数据统计**：仪表盘展示阅读时长、笔记趋势、常用标签等，可视化你的阅读成长。
- 🎨 **多主题切换**：内置森绿、海蓝、暖橙、薰衣草四种水彩主题，背景随主题动态切换。
- 🗑️ **回收站**：软删除机制，可恢复误删的书籍与笔记。

---

## 🛠️ 技术栈

- **前端**：React + TypeScript
- **UI 组件**：Ant Design、Framer Motion
- **数据可视化**：Recharts
- **后端与数据库**：Supabase（PostgreSQL） / 本地存储（localStorage）双模式
- **构建工具**：Vite
- **部署**：Vercel
